### Scene. 메인

- 구글 연동 로그인 가능, 로그인 이력 저장
- 현재 스킨 표시
- 상점 씬 연결
- 설정: 효과음, 배경음 조절/ 로그아웃
- 온라인 매칭 -> 매칭 성공 시 게임 씬 연결
<br>

### Scene. 상점
- 현재 재화량, 스킨 표시
- 스킨 목록(이름, 외형, 가격) 표시 (DB에서 관리)
- 구매 시 DB의 유저 인벤토리,재화량 업데이트 & 자동 장착
